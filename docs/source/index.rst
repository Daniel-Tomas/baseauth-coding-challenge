Welcome to baseauth's documentation!
===========================================

baseauth is a user authentication and preferences system, based on the CAS protocol,
implemented with Django. baseauth is used as the central authentication component
for a range of base applications. It can be used with other authentication backends
and identity providers based on LDAP, Shibboleth, or other CAS servers.
